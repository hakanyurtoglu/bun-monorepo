# Bun Monorepo

This is a monorepo project that contains a backend built with Hono and a frontend built with React, Vite, and Tailwind CSS.

## Getting Started

**Clone the repository**

**To install dependencies:**

```bash

bun install

```

**To run (both backend and frontend) :**

```bash

bun dev

```

**Run them seperately:**

```bash

bun dev:web

bun dev:backend

```
